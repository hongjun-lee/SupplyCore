### §3.6 微信小程序 Wave 2 风险防御

| 风险 | 防御 | 触发回退条件 |
|---|---|---|
| **微信订阅消息审核失败** | 模板申请前 cici 审核 / 备用模板 ≥ 2 / 审核失败立修 | 任 1 模板审核失败 → 修改重提 / 不阻塞 Wave 2 启动 |

| **跨模块待办聚合性能** | 后端 API 缓存 + 限频 + 分页 / 移动端骨架屏 / 不阻塞首屏 | API 响应 ≥ 3s → 立修缓存策略 |
| **审批推送频率骚扰** | 推送频率限制（每天最多 5 条）+ 用户可关闭 + 静默时段（22:00-08:00）| 用户反馈骚扰 → 立修推送策略 |
