**责任部门**：物资公司 + 各厂矿物资部
**填报方式**：Excel 模板 + 系统导入 endpoint POST /api/supply-cores/data-import/material

## 二、物料编码（Sheet 2: Material）

### 2.1 字段清单

| # | 字段 | 类型 | 长度 | 必填 | 说明 / 校验 |
|---|---|---|---|---|---|
| 1 | 物料编码 | material_code | string | 32 | ✅ | 唯一 / 按编码规则 / 例 `M-01-01-0001` |
| 2 | 物料名称 | material_name | string | 128 | ✅ |  |
| 3 | 规格型号 | specification | string | 256 | ✅ | 例 `YJV22-3×95+1×50` |
| 4 | 计量单位 | unit | string | 16 | ✅ | 必须在 Unit sheet（基础单位）|
| 5 | 分类编码 | category_code | string | 16 | ✅ | 必须在 MaterialCategory sheet（二级或以下）|
| 6 | NC 物料编码 | nc_material_code | string | 32 | ⚠️ | NC 系统物料编码 / 双号制对照 / 缺失记 NC 映射台账 |
| 7 | 是否高敏感 | is_high_sensitive | bool | - | ⚠️ | 默认从分类继承 |
| 8 | 是否设备备件 | is_equipment_spare | bool | - | ⚠️ |  |
| 9 | 是否可采购 | is_purchasable | bool | - | ⚠️ | 默认 true |
| 10 | 是否启用 | is_active | bool | - | ⚠️ | 默认 true |
| 11 | 安全库存 | safety_stock | decimal | - | ❌ | 物资公司维护 |
| 12 | 备注 | remark | string | 256 | ❌ |  |

### 2.2 示例

| material_code | material_name | specification | unit | category_code | nc_material_code | is_high_sensitive |
|---|---|---|---|---|---|---|
| M-01-01-0001 | 电力电缆 | YJV22-3×95+1×50 | 米 | 01-01 | NC-CABLE-001 | false |
| M-02-01-0001 | 乳化炸药 | 32mm × 200mm | 卷 | 02-01 | NC-EXP-001 | **true** |
| M-03-0001 | 输送带 | B=1000mm | 米 | 03 |  | false |
| M-04-0001 | 安全帽 | 国标 | 顶 | 04 | NC-PPE-001 | false |

### 2.3 校验规则

- `material_code` 唯一性
- `material_name + specification` 组合唯一（防重复物料不同编码）
- `unit` 必须在 Unit sheet
- `category_code` 必须为二级或以下（避免一级直接挂物料）
- `nc_material_code` 缺失 → 记 NC 映射台账
- `is_high_sensitive` 若与分类不一致 → 警告 + 记台账

## 三、计量单位（Sheet 3: Unit）

### 3.1 字段清单

| # | 字段 | 类型 | 必填 | 说明 |
|---|---|---|---|---|
| 1 | unit_code | string | ✅ | 唯一 / 例 `M` `KG` `T` `件` `卷` |
| 2 | unit_name | string | ✅ | 中文显示 / 米 / 千克 / 吨 |

| 4 | base_unit_code | string | ⚠️ | 转换基础单位（吨 → 千克）|

## 四、导入与错误反馈

### 4.1 系统导入流程

同 `01-组织与人员` §四 / type=`material`

### 4.2 关键校验聚焦

- **物料编码重复**：高优先级 / 记台账 / 业务方确认是否合并
- **NC 物料编码缺失 / 不一致**：必记 NC 映射台账 / cici 与 NC 侧对照
- **高敏感属性误标**：火工品 / 易燃易爆 / 必双重确认
- **计量单位错误**：影响库存数量准确性 / 严格校验

### 4.3 数据问题台账（物资专属）

| 类型 | 描述 | 责任部门 | 严重度 |
|---|---|---|---|
| material_code_duplicate | 编码重复 | 物资公司 | 高 |
| material_no_nc_mapping | NC 编码缺失 | 物资公司 + 财务 | 中 |
| high_sensitive_mismatch | 高敏感属性与分类不一致 | 物资公司 | 高 |
| invalid_unit | 计量单位错误 | 物资公司 | 高 |
