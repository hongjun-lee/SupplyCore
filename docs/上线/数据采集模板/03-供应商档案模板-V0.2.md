# 03 供应商档案数据采集模板 V0.2（cici 反馈 #3+#4 修订 — 链接同步 V0.2 / 2026-05-17）

**目的**：第 1 批基础数据采集 / 供应商基础信息 + 资质 + 财务 + NC 对照
**责任部门**：物资公司供应商管理 + 财务
**填报方式**：Excel 模板 + 系统导入 endpoint POST /api/supply-cores/data-import/supplier
**数据源优先**：现有供应商库迁移 + NC 供应商对照 + 资质审核记录

## 一、供应商基础（Sheet 1: Supplier）

### 1.1 字段清单

| # | 字段 | 类型 | 长度 | 必填 | 说明 / 校验 |
|---|---|---|---|---|---|
| 1 | 供应商编码 | supplier_code | string | 32 | ✅ | 唯一 / 按编码规则 / 例 `SUP-202601-0001` |
| 2 | 供应商名称 | supplier_name | string | 256 | ✅ | 与营业执照一致 |
| 3 | 统一社会信用代码 | unified_credit_code | string | 18 | ✅ | 18 位 / 唯一 |
| 4 | 法定代表人 | legal_representative | string | 64 | ✅ |  |
| 5 | 注册资本（万元）| registered_capital | decimal | - | ⚠️ |  |
| 6 | 注册地址 | registered_address | string | 256 | ⚠️ |  |
| 7 | 经营范围 | business_scope | string | 1024 | ⚠️ |  |
| 8 | 供应商类型 | supplier_type | enum | 32 | ✅ | 制造商 / 经销商 / 代理商 / 服务商 / 其他 |
| 9 | 准入状态 | access_status | enum | 16 | ✅ | 待审核 / 已准入 / 暂停 / 黑名单 |
| 10 | 准入日期 | access_date | date | - | ⚠️ |  |
| 11 | 黑名单原因 | blacklist_reason | string | 256 | ⚠️ | 黑名单时必填 |
| 12 | 是否启用 | is_active | bool | - | ⚠️ | 默认 true |
| 13 | 备注 | remark | string | 512 | ❌ |  |

### 4.2 临期预警

- 系统每日 cron job 检查 expire_date
- 30 天内到期 → ReportAlert 推送 + dashboard 高敏感关注
- 已过期 → 供应商 access_status 自动改为"待审核" + 通知 supplier 管理员

## 五、NC 供应商映射（Sheet 5: NcSupplierMapping）

### 5.1 字段清单（关键 / 双号制核心）

| # | 字段 | 类型 | 必填 | 说明 |
|---|---|---|---|---|
| 1 | supplier_code | string | ✅ | SupplyCores 供应商编码 |
| 2 | nc_supplier_code | string | ✅ | NC 系统供应商编码 |
| 3 | mapping_status | enum | ✅ | 已映射 / 待确认 / 冲突 |
| 4 | mapping_notes | string | ❌ | 备注（冲突原因 / 历史合并等）|

### 5.2 缺失或冲突处理

- 缺失（supplier_code 无 nc_supplier_code）→ 记 NC 映射台账 / cici 与 NC 侧对照
- 冲突（同 nc_supplier_code 对应多 supplier_code 或反之）→ 严重 / 人工审核

## 六、导入与错误反馈

### 6.1 关键校验

- `supplier_code` 唯一
- `unified_credit_code` 唯一（18 位国标）+ 与营业执照一致
- 每供应商至少 1 个主联系人 / 1 个主账户
- 资质有效期检查 / 临期供应商标记
- NC 供应商映射完整性（缺失 / 冲突 严重度高）

### 6.2 数据问题台账（供应商专属）

| 类型 | 描述 | 责任部门 | 严重度 |
|---|---|---|---|
| supplier_no_nc_mapping | NC 供应商编码缺失 | 物资 + 财务 | 高 |
| supplier_qualification_expired | 资质已过期 | 物资 | 高 |
| supplier_qualification_near_expiry | 资质 30 天临期 | 物资 | 中 |
| supplier_duplicate_credit_code | 信用代码重复（可能合并）| 物资 | 高 |
| supplier_blacklist_no_reason | 黑名单无原因 | 物资 | 中 |

## 七、典型工作量预估

- 集团供应商数：~200-500 家
- 填报时间：3-5 天（物资公司供应商管理）
- 校验 + 清洗：5-7 天（物资公司 + 财务）
- NC 映射对照：3-5 天（cici + NC 管理方）
