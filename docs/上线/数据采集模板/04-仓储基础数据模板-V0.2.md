## 一、仓库（Sheet 1: Warehouse）

### 1.1 字段清单

| # | 字段 | 类型 | 长度 | 必填 | 说明 / 校验 |
|---|---|---|---|---|---|
| 1 | 仓库编码 | warehouse_code | string | 32 | ✅ | 唯一 / 按编码规则 / 例 `WH-FK002-001` |
| 2 | 仓库名称 | warehouse_name | string | 128 | ✅ | 例 `艾友矿主仓库` |
| 3 | 所属组织编码 | org_code | string | 32 | ✅ | 必须在 Organization sheet（厂矿级或下）|
| 4 | 仓库类型 | warehouse_type | enum | 32 | ✅ | 主仓 / 临时仓 / 工地仓 / 库外保管 / 报废仓 |
| 5 | 仓库地址 | address | string | 256 | ⚠️ |  |
| 6 | 仓库管理员工号 | manager_employee_no | string | 32 | ✅ | 必须在 User sheet / role 仓储 |
| 7 | 联系电话 | phone | string | 16 | ⚠️ |  |
| 8 | 启用日期 | active_date | date | - | ✅ |  |
| 9 | 是否启用 | is_active | bool | - | ⚠️ | 默认 true |
| 10 | 是否启用批次 | enable_batch | bool | - | ⚠️ | 火工品 / 易燃易爆必启 |
| 11 | 是否启用序列号 | enable_serial | bool | - | ⚠️ | 设备 / 高价值必启 |
| 12 | 是否启用有效期 | enable_expiry | bool | - | ⚠️ | 化学品 / 油料必启 |
| 13 | 备注 | remark | string | 256 | ❌ |  |

## 四、导入与错误反馈

### 4.1 关键校验

- `warehouse_code` 唯一 + org_code 必须在 Organization
- `manager_employee_no` 必须在 User sheet + role 仓储
- `zone_code` 仓库范围内唯一
- `location_code` 全表唯一 + zone_code + warehouse_code 一致
- `enable_batch / enable_serial / enable_expiry` 按 warehouse_type 推荐（火工品仓必启 batch + expiry）
- `freeze_reason` 冻结状态必填

### 4.2 数据问题台账（仓储专属）

| 类型 | 描述 | 责任部门 | 严重度 |
|---|---|---|---|
| warehouse_no_manager | 仓库无管理员 | 各厂矿仓储 | 高 |
| warehouse_high_sensitive_no_batch | 火工品仓未启批次 | 物资 + 各厂矿 | 高 |
| location_status_freeze_no_reason | 冻结无原因 | 各厂矿仓储 | 中 |
| zone_warehouse_mismatch | 库区仓库不一致 | 各厂矿仓储 | 高 |

## 五、典型工作量预估

- 集团仓库数：~20-50 个（5 厂矿 + 本部 + 临时仓 + 工地仓）
- 库区数：~100-300
- 货位数：~5000-20000
- 填报时间：5-7 天（各厂矿仓储）
- 校验 + 清洗：3-5 天（物资公司）
