## 三、关键合规对齐（集团统一建设原则 V2.0 落地）

- **统一身份**：对接集团 Nova Platform SSO，组织/人员/数据权限以平台为权威源
- **统一接口**：对外 API + JSON 标准；集团统一 API 网关由网信办统筹建设，本系统按网信办提供规范接入
- **权威数据源**：组织人员→平台，会计科目→NC，物料编码→本系统，供应商/合同/资产→预留对齐
- **信创合规**：须采用 PostgreSQL 兼容国产数据库（瀚高 HGDB、人大金仓 KingbaseES 等）
- **AI 入口**：必做要求，统一接入网信办 AI 入口，不得另建独立门户
- **NC 协同**：N:1 映射、物资适配 NC（NC 零改造），严格反结、完整对账
