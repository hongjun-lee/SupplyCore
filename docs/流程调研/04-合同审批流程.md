# 采购合同流程

> **来源：** `docs/流程调研/调研原文档/4.合同审批流程(按新表序调整).docx`
> **范围：** 业务员起草合同 → 三路审批分流（内部 / 20万以下 / 20万以上）→ 供应商盖章 → 加盖合同章 → 合同编号登记 → 履约保证金（条件触发）
> **起草依据：** 中标通知 / 成交通知书

---

## 1. 内部企业审批链（合同备案）

> 触发：交易对方为集团内部企业（无金额阈值）

```mermaid
flowchart LR
    A[供应商签字盖章] --> B[专业/技术部门审核]
    B --> C[合同管理部门审核]
    C --> D[财务部门审核]
    D --> E[分管领导<br/>总会计师 会签]
    E --> F[单位负责人 会签]
```

- 入口表单：**《阜矿集团所属单位合同备案单》**
- 审批链最短，仅走物资公司内部会签

## 2. 20 万以下审批链（小额会签）

> 触发：交易对方非内部企业，且合同金额 ≤ 20 万元

```mermaid
flowchart LR
    A[供应商签字盖章] --> B[专业/技术部门审核]
    B --> C[合同管理部门审核]
    C --> D[财务部门审核]
    D --> E[分管领导<br/>总会计师 会签]
    E --> F[单位负责人 会签]
```

- 入口表单：**《物资公司合同会签单》**
- 与内部企业链条节点相同，但入口表单不同（**物资公司层面**而非集团所属单位）

## 3. 20 万以上审批链（集团多部门会签）

> 触发：交易对方非内部企业，且合同金额 > 20 万元

```mermaid
flowchart TD
    A[供应商签字盖章] --> B[专业/技术部门审核]
    B --> C[合同管理部门审核]
    C --> D[财务部门审核]
    D --> E[分管领导/总会计师 会签]
    E --> F[单位负责人 会签]
    F --> G[集团公司财务部、法控部、<br/>经发部 会签]
    G --> H[集团公司分管领导 会签]
    H --> I[集团公司总法律顾问 会签]
    I --> Q5{合同金额<br/>是否超过 1000 万?}
    Q5 -- 否 --> End([进入合同章/编号登记])
    Q5 -- 是 --> J[集团公司总经理 会签] --> End
```

- 入口表单：**《阜矿集团所属单位合同会签单》**
- 物资公司内部会签 + **集团层面四级会签**（财务/法控/经发 → 分管 → 总法律顾问 → 总经理）
- **1000 万阈值**触发集团公司总经理会签

---

## 4. 汇合后动作

| 顺序 | 动作 | 责任方 |
|---|---|---|
| 1 | 业务员向合同管理部门申请加盖合同章 | 业务员 + 合同管理部门 |
| 2 | 业务员向企划部申请合同编号，登记纸质台账并签字（表单） | 业务员 + 企划部 |
| 3 | （条件）签订履约保证金协议、缴纳履约保证金 | 业务员 + 供应商 |
