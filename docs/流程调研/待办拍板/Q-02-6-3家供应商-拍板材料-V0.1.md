# Q-02-6「3 家以上供应商」是软提示还是强约束 — 拍板材料

> **状态：** ✅ 已拍板（2026-05-09，方案 C：强约束 + 简化特批）
> **场景：** 询比 / 竞价采购少于 3 家供应商时的系统行为
> **背景：** 业务方答复**软提示**，政策规定**硬性 ≥ 3 家**，两者冲突

---

## 一、问题背景（30 秒）

| 维度 | 详情 |
|---|---|
| **业务场景** | 物资采购走"询比采购"或"竞价采购"非招标方式时，参与报价 / 响应的供应商数量 < 3 家 |
| **触发频度** | 紧急采购 / 独家供应 / 集团目录内有限供应商 / 防爆煤安专项资质等场景较高频 |
| **影响范围** | 详设 04 询价/谈判模块 + 详设 09 ALR-PUR-001 报表 + 详设 10 审批模板 |

---

## 二、业务方答复 vs 政策规定 — 冲突点

### 2.1 业务方答复（P0 Q-02-6，2026-05-09）

> **(x) 软提示**（警告但允许提交）

### 2.2 政策规定（政策 04 采购管理办法 第十九条）

> 询比采购、竞价采购**同时满足下列情形的可采用**：
> 1. 采购需求清晰、准确、完整
> 2. 技术和质量标准化程度较高
> 3. **市场资源较丰富、潜在供应商不少于 3 家**
> 4. 应急采购

> **解读：** 政策"3 家以上"是 4 个判定条件之一（属硬性条件），不满足应转为其他方式（直接采购等）而非"软提示放行"。

### 2.3 冲突本质

业务方实操：3 家不足时仍走询比/竞价但**实际只有 1-2 家响应**
政策口径：3 家不足时**应转直接采购**（条件 5 之一：有效供应商有且仅有 1 家）

---

## 三、3 个方案对比（带利弊 + 推荐）

### 方案 A：严格按政策 — 硬约束 + 自动转直接采购

```
< 3 家供应商响应 → 系统拦截 → 强制业务员转"直接采购"路径 → 走直接采购的 8 种适用情形之一审批
```

| 利 | 弊 |
|---|---|
| ✅ 完全合规，审计无风险 | ❌ 操作繁琐，业务员需重新走直接采购流程 |
| ✅ 系统强约束防违规 | ❌ 应急场景延迟（重启流程）|
| ✅ 与政策一致，无解释成本 | ❌ 与业务方实操习惯不符 |

### 方案 B：业务方实操 — 软提示放行

```
< 3 家供应商响应 → 警告（"建议补充供应商至 3 家以上"）→ 允许提交 → 备注栏说明原因 → 详设 09 ALR-PUR-001 报表暴露
```

| 利 | 弊 |
|---|---|
| ✅ 操作灵活、不阻断 | ❌ 合规风险（审计可能挑战）|
| ✅ 与业务方习惯一致 | ❌ 滋生"为达成而忽略 3 家底线" |
| ✅ 应急场景效率高 | ❌ 月度报表仅事后暴露不足比例 |

### 方案 C：折中（**推荐 ⭐**）— 强约束 + 简化特批

```
< 3 家供应商响应 → 系统拦截 → 业务员填《不足 3 家说明》走简化特批
                                    ├─ 物资公司业务部主管 + 财务复核（非集团层面）
                                    └─ 留痕入 A-14 审批日志
```

| 利 | 弊 |
|---|---|
| ✅ 合规可追溯（A-14 留痕）| ⚠ 系统复杂度提升（新增特批路径）|
| ✅ 简化特批（不上集团）→ 业务效率不大降 | ⚠ 需 Spencer 与业务方共同确认特批层级 |
| ✅ 平衡政策刚性与实操弹性 | |

## 五、推荐方向 + 拍板要点

### 推荐 ⭐ 方案 C（强约束 + 简化特批）

**理由：**
1. **合规与效率平衡** — 既不让业务员"自由放行"（方案 B 风险），也不让业务员"重启流程"（方案 A 阻断）
2. **审计可追溯** — 所有不足 3 家的特批走 A-14 审批日志
3. **改造成本最小** — V0.2 已落 ALR-PUR-001，加 1 个简化特批模板即可
