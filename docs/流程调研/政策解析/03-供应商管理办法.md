# 阜新矿业集团物资有限公司 供应商管理办法（试行）

> **来源：** `docs/流程调研/调研原文档/3.供应商管理办法.pdf`
> **附表：** `03A-供应商管理办法附表.md`（5 张表单：推荐表/新增审批表/反馈单/评价表/年度考评表）
> **重大价值：** **本办法回答了 P0 中 Q-03 系列多项疑点**（5 级分类标准、评价百分制、冷却期、第三方推荐、试用期等）—— 业务方 Q-03-2 答"其他完全方案"实际就是本办法。

---

## 一、适用范围与组织架构

| 维度 | 内容 |
|---|---|
| **适用主体** | 阜新矿业集团物资有限公司（"物资公司"） |
| **管理对象** | 为物资公司提供货物和服务的**所有供应商** |
| **领导小组** | 组长 = 经理；副组长 = 副经理 / 总会计师 / 副总经济师 / 副总工程师 |
| **业务中心** | 信息收集 + 调查；资信 / 技术 / 质量 / 履约能力评估；选择推荐潜在供应商 |
| **企划管理中心** | 动态完善供应商信息库；汇总整理档案 |

---

## 二、准入原则（六项）

1. 公开、公平、公正
2. 广泛吸收、严格准入、有效竞争、优胜劣汰
3. **同质同价、内部单位优先**
4. 以**生产厂商为主**，代理商和贸易商为辅
5. 产品使用技术成熟、质量稳定可靠优先
6. **有知名度、良好业绩**企业优先

## 三、供应商引进与审核流程


> **试用期：** 成交供应商有 **1 年试用期**；试用期内没有再发生业务或评价为 **D 级及以下**的供应商应**取消其供应商资格**。
> **履约保证金：** 核准引进的供应商供货时，**如有必要可缴纳一定比例履约保证金**（具体比例本办法未明示）。

## 四、供应商分级管理（重大！P0 联动）

### 4.1 三大级 × 五小级

> **大级分类：** 潜在 / 合格 / 不合格
>
> **合格再细分（按年度评价）：** 优质 / 重点 / 一般 / 高风险 / 不合格

| 大级 | 定义 | 来源 |
|---|---|---|
| **潜在供应商** | 公开招标符合投标条件但未中标 + 第三方推荐 | 第十三条 |
| **合格供应商** | 通过合理采购方式确定的合作关系，质量/价格/供货时间/售后均达使用要求 | 第十四条 |
| **不合格供应商** | 不履行采购合同 / 提供产品不达标 / 造成重大经济损失 / 其他特定情形（8 种） | 第十五条 |

### 4.2 年度评价等级（百分制）

| 等级 | 分数 | 类别要求 |
|---|---|---|
| **A 级** | ≥ 90 分 | 优质供应商必须达 A |
| **B 级** | 80-89 分 | 重点供应商达 B 及以上；若评 C 降为一般 |
| **C 级** | 70-79 分 | 一般供应商达 C 及以上；若连续 2 年 A 可升重点 |
| **D 级** | 60-69 分 | **高风险供应商** |
| **E 级** | < 60 分 | **不合格供应商** |

**晋升规则：**
- 重点连续 2 年 A → 升优质（同类产品）
- 一般连续 2 年 A → 升重点

**降级规则：**
- 优质评 B → 降重点
- 重点评 C → 降一般
- 内供品质量问题 / 售后不及时 → 经济处罚 + 上报省能源集团 → 严重者取消内供资格

### 4.3 评分维度（4 项 + 累计扣分规则）

> 评价采用百分制，**4 个维度**：
> - **产品质量**
> - **产品价格**
> - **履行能力**
> - **售后服务**

**累计扣分规则：**
- 同一供应商同类产品在不同项目中出现扣分 → 累计扣分，**该项分值扣完为止**
- 同一供应商同类同批次产品多个单位或部门扣分 → **不累加**计算
- 各考评分项分值扣完为止，**不越项、不超分值**

## 五、停止供应与永久退出

### 5.1 立即停止供应业务（第二十一条）

5 种情形 → 立即停止业务，重新启用需经审批：
1. 评定不合格需整改
2. 资质过期不更新
3. **2 年内不产生业务**
4. 产品出现一次供应问题且没造成损失
5. 不服从供应系统统一管理

### 5.2 不合格供应商重启规则（关键！）

> **不合格供应商，原则上一年内不准参与公司的物资采购，如需参加需要重新审批。**

### 5.3 永久退出（3 种情形，无法重审）

1. 提供虚假资料骗取资格
2. 向采购人员行贿或提供不正当利益
3. 与采购人员串通虚高价格或提供虚假产品

## 六、档案与采购联动

| 规则 | 内容 |
|---|---|
| **档案库结构** | 供应商信息库分 3 类：潜在 / 合格 / 不合格 |
| **管理形式** | "供应商为纲、物资为目"列表形式 |
| **动态更新** | 根据采购情况 + 使用单位反馈 + 考核评估结果动态更新 |
| **非招标采购** | 优先从**合格供应商、潜在供应商**中选取邀请对象 |
| **优先级** | 其他条件相同时，**优先选优质 / 重点供应商** |
| **档案查阅** | 须先向档案管理部门申请 |
