## 一、可提取的关键信息（来自附件 1 备案表）

### 1.1 招租方式（共 3 种）

| 招租方式 | 说明 |
|---|---|
| **公开挂牌** | 通过挂牌平台公开发布租赁信息 |
| **其他公开招租方式** | 公开招租但非挂牌渠道 |
| **协议** | 协议直租（与特定承租人协商） |

### 1.2 资产租赁备案表必填字段

| 字段类别 | 字段 |
|---|---|
| **资产基本信息** | 资产名称 / 资产类别 / 地址 / 资产面积 / **资产原值** / **资产净值** |
| **资产状态** | 有无产权 / 资产现状 |
| **租赁信息** | 招租方式 / 承租人 / 租赁用途 / 租赁期限（年月日 → 年月日） |
| **金额** | 租金 / **保证金** |

### 1.3 多部门审核栏（备案表签字栏）

```
申报单位（公章）→ 申报单位负责人 → 生活服务中心 → 财务部 → 安监部 → 武保部 → 经发部
```

> **6 部门会签**：生活服务中心 / 财务部 / 安监部 / 武保部 / 经发部 + 申报单位负责人

## 二、可提取的关键信息（来自附件 2 房屋租赁合同模板）

> 合同模板覆盖标准条款（约 16 节），**作为政策附件主要价值**：

| 节 | 关键内容 |
|---|---|
| 租金、递增、保证金 | 租金标准 + 逐年递增 % + 期次（按月/季）+ 银行转账方式 + **首期预付租金 + 租赁保证金** |
| 租赁期间费用 | 甲方：房屋出租税费；乙方：水电气暖电话有线电视宽带物业等 |
| 修缮与使用 | 甲方提前 15 天书面通知 + 乙方合理使用 + 不改主体结构 |
| 转租限制 | **未经甲方书面同意不得转租 / 转借** |
| 解除合同 | 拖欠房租超 N 个月 / 转租 / 损坏 / 违法活动 / 安全隐患整改不到位 |
| 验收 | **租赁前 + 租赁期满双重验收**，照片 / 图纸 / 影像作合同附件 |
| 违约责任 | 甲方违约 % / 乙方违约 % / 逾期租金日费 % |
| 保密条款 | 商业信息 / 技术信息 / 缔约条件等 |

## 三、与 P0 答复 / 业务关联

| 维度 | 关联 |
|---|---|
| **Q-00-1 出租设备业务** | ⚠️ 本细则是"**资产租赁**"（房屋 / 场地 / 建筑物），**不是设备出租**；业务方 Q-00-1 答复提交的是"检修管理办法"（政策 01），与本细则不同 |
| **0-内控总体框架图 §出租设备** | 仍未明确 — 可能指设备租赁（如租用外部设备）or 出租自有设备给外部 |
| **资产租赁是否纳入 SupplyCore 一期** | 待业务方决议 — 本细则覆盖的是物资公司持有不动产对外出租（房屋 / 场地），与"物资管理"业务模块关联弱 |

## 四、是否深入解析的建议

| 选项 | 理由 |
|---|---|
| **A. 暂停本细则的深入解析（推荐）** | 1. 正文（前 12 页）扫描版需 OCR<br/>2. "资产租赁"与物资公司核心业务（采购/库存/合同）关联弱<br/>3. 合同模板字段已显式提取，备案表字段已记录 |
| B. OCR 解析正文 | 工作量大，价值有限；如确认要做需用 OCR 工具（macOS Vision / Tesseract） |
| C. 等业务方澄清 Q-00-1 后再决定 | 如业务方确认"出租设备"含资产租赁，则需深入解析；否则归档 |

> **当前结论：** 资产租赁不属于物资公司核心业务范畴，本细则**作为政策档案保留即可**，不阻塞详设。
