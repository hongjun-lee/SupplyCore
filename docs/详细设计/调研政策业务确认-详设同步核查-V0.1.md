# 调研政策业务确认-详设同步核查（V0.1）

> 核查日期：2026-05-10  
> 范围：近期业务方 P0 答复、政策解析 01-06、流程调研新增流程 14 与现有详细设计 01-11 的同步关系。

## 一、结论

需要同步变更。采购阈值、供应商分级、合同会签、履约保证金、暂估闭环、出库打印、委托加工和超储处置等主线已经大体进入 04 / 05 / 06 / 09 / 10；本轮从头核查后，发现并补齐 5 类会影响开发口径的缺口：

1. S-01 采购申请缺少业务流向字段，无法区分外购入库 / 直达 / 代储 / 外委检修 / 委托加工。
2. 非直达物资临时走直达通道的审批层级（业务主管 / >100 万总经理）未落到 04 / 10。
3. 外委检修由政策和流程 14 补全后，07 / 10 / 05 / 11 需要补审批、价格上限、附件、SLA 和配置口径。
4. 履约保证金“没收”已在 05 识别，但 10 高敏感清单缺 SENS-CON-003。
5. 11 的暂估配置仍残留单一 30 天预警，应与 06 / 09 的 D-90 / D-30 双预警保持一致。
