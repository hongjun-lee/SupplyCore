# AI 接入专题说明

## 一、定位

本目录用于归集“业务系统并入集团统一 AI 入口”的专题材料，作为集团统筹文档下的专项参考资料目录。

其中：

- 集团层面的通用原则，统一写入：
  - `docs/集团统筹/集团业务系统统一建设原则-V2.0.md`
- 各业务系统的专项 AI 接入要求、招标条款、Tool 清单、验收条款等材料，按系统分别归档在本目录下。

## 二、当前已归档内容

### 物资管理系统

路径：`docs/集团统筹/AI接入专题/物资管理系统/`

当前包含：

1. `01-并入统一AI入口能力要求（建议稿）.md`
2. `02-招标正文精简版.md`
3. `03-接口及Tool清单（附件建议稿）.md`
4. `04-验收条款表（建议稿）.md`

## 三、使用建议

- **集团统筹文件** 写“通用原则”
- **系统专项文件** 写“具体接入要求、能力清单、联调要求、验收条款”
- 后续若财务、人资、设备、生产等系统也需要统一 AI 接入要求，可继续按相同目录结构扩展

## 四、一句话说明

本目录解决的是“同一套统一 AI 入口原则，如何在不同业务系统中形成可复用、可归档、可扩展的专项材料”。
