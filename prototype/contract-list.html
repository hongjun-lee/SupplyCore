⚖ C-02 8 状态机详细规则（详设 05 §4.2.1）
1️⃣ 草稿 → 会签中：采购员发起会签 (法务 + 财务 + 业务负责人)
2️⃣ 会签中 → 已签：会签全部通过 + 双方电子签章 (NC-MD-007 同步)
3️⃣ 会签中 → 草稿：会签驳回，回滚至草稿态 (允许修改后重提)
4️⃣ 已签 → 执行中：合同生效日到 / 首笔订单触发
5️⃣ 执行中 → 已完成：验收 + 付款 + 履约保证金退还全部完成
6️⃣ 执行中 → 已变更：履行期间金额 / 期限 / 范围调整 (变更需按原会签流重走)
7️⃣ 已变更 → 执行中：变更生效后回到执行态
8️⃣ 执行中 → 已终止：违约 / 协商终止 (履约保证金按 SENS-CON-003 处理)
