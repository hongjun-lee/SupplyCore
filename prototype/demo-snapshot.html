⚖ 数据快照 / 导入导出规则
使用场景：
• 跨设备演示：A 设备保存快照 → JSON 文件传到 B 设备 → 导入还原 → 1:1 重现演示场景
• 多场景切换：保存"采购入库主线-演示完整"、"暂估闭环-演示中"、"流标场景-演示完整"等多个快照，按需切换
• 培训演示：保存培训用初始快照，每次开课前还原
• 故障恢复：演示中误操作 → 还原到上次保存点
边界与限制：
• LocalStorage 配额限制（5 MB），单浏览器最多保存 ~5-8 个快照
• JSON 导入会完全替换当前数据，不会合并
• 快照不含图片 / 文件附件（仅数据）
• 快照与代码版本绑定 — 升级到不兼容的代码后旧快照可能加载失败
