指标定义与回写来源
  • 运行率 = (计划运行时长 − 非计划停机) / 计划运行时长；非计划停机来自 E-04 维修工单 的"停机时长"字段（关单后回写）
  • 性能率 = 实际产出 / 理论产出；产出从生产端集成（当前为人工录入或月度核算，后续对接生产系统）
  • 良品率 = 1 − 设备相关返工损失率；与 03 物料质检相关字段联动
  • 预警阈值 = OEE < 60% 触发 09 报表预警，自动推送 → 维修班长 + 设备管理员
  • 统计粒度 = 日 / 周 / 月，存于 equipment_oee_daily / weekly / monthly（详设 07 + 09）
边界说明
本看板为 设备综合效率分析视图，不替代生产系统的 OEE 主数据。
当前：性能率与良品率支持人工录入或 CSV 导入；后续对接厂矿生产系统自动取值。
OEE 数据用于 设备管理决策与维修优先级，不用于 KPI 考核口径，避免误用。
📋 页面说明 · 业务依据 + 实现要点
📋 业务依据：详设 09 V1.2 §六 RPT-EQ-OEE 设备综合效率报表 + 详设 07 V1.0a §设备运行。
📐 OEE 公式：OEE = 可用率 × 性能率 × 良品率；可用率 = (计划 - 故障停机 - 计划停机) / 计划；性能率 = 实际产量 / (理论速度 × 实际工时)；良品率 = 合格 / 总产量。
🛤 链路：E-01 设备状态变化 → 累计停机时长 → linkage 自动重算 OEE → R-05 预警（OEE < 阈值 / 故障停机 > 24h）→ 月度报表 RPT-EQ-OEE。
