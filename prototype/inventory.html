📋 页面说明 · 业务依据 + 实现要点
业务依据：详设 06 V1.1a §库存查询主页 + S-13 库存台账（实时数量/金额）+ S-14 批次账（FIFO/批次追溯）+ S-21 库存事务流水（不可修改）。
链路：所有出入库 / 调拨 / 盘点 / 废旧 → 同事务原子写 S-13 + S-14 + S-21 → 触发 F-01 NC 任务 → 库存查询页实时聚合显示；穿透到 S-21 流水可看每一笔变动来源。
高敏感：火工品 / 危险化学品库存仅授权角色可见；月末结账后冻结期内库存不允许修改（详设 06 §6.4）。
实现说明： · 库存唯一事实源原则：所有库存数量与价值以 S-13 库存台账 为准；批次维度信息见 S-14 批次账；任意一笔出入库须在同事务原子写入 S-21 库存事务流水。 · 低储 / 超储口径：低库存按物料档案 safety_stock；超储按储备资金预算 110% + 安全周转周期双维度，不再使用单一最高库存字段。 · 火工品 / 化工品强制批次：分类前缀 HG/HX 必须按批次维度记账，并支持双人双锁专管库。
