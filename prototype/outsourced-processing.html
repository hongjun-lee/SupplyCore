🏭 受托虚拟仓 + 标准损耗率
受托虚拟仓：原料投料后挂在受托方虚拟仓（不在物资公司主库），便于追踪在途原料和产成品。
标准损耗率：按物料类别配置；超过损耗率需说明原因（例：金属切削加工 ≤ 5%，化工分馏 ≤ 8%）。

👥 三方联合验收
委托方（物资公司）：核对原料投料数量 + 加工费金额 + 损耗率
受托方（加工厂）：提供加工记录 + 损耗清单 + 产成品质检报告
使用单位（终端）：验收产成品物理状态 + 数量 + 入实物账
验收要求：三方签字（电子签章）+ 拍照留痕 + 必要时第三方检验报告（如金属硬度、化工纯度）
异常处理：损耗率超阈值 → 受托方需提供书面说明 + 物资公司补缺料或核减加工费
