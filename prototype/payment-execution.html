🔁 NC 实付回写时序
1. C-08 终审通过 → C-10 自动生成（state=待执行）
2. 三单匹配前置校验 → 通过 → state=已校验
3. 推送 BIZ-020 NC 接口 → state=推送中
4. 等待 NC 回执（payment_voucher_no + actual_amount + actual_date）
5. 收到 NC 回执 → state=已支付，回写：
C-07.cumulative_paid_amount += actual_amount
C-07.remaining_amount -= actual_amount
C-02.paid_amount += actual_amount
6. 检查 C-07.remaining ≤ 0 → C-04.node_state=已付款
7. 检查 C-02.paid ≥ contract_amount × (1 - retention) → C-02 → 已完成
8. 失败回退：state=支付失败 → C-08 退回到"支付退回"重发起
