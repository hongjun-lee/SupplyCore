📋 规则依据：政策 04《阜新矿业集团采购管理办法》（阜矿发 [2025] 70 号）第二十六条："任何单位和个人不得将本办法规定的招标项目化整为零或以其他任何方式规避招标"+ 第十七条"同类合并：年度累计达到上述应招标限额的，应合并招标或采取入围招标"。
🛡 检测窗口：30 天滚动窗口（业务方答 Q-02-1 已落 V0.3）；按 (部门 × 物料类别 × 供应商类别) 三维分组累计采购申请金额，达到应招标阈值时触发 ALR-PUR-SPLIT-001。
⚠ 化整为零可疑分组（30 天滚动窗口）
分组键 部门 物料类别 申请笔数 累计金额 应招标阈值 超阈值 风险级别 操作
检测逻辑（伪代码）
FOR each (dept, material_category) 分组:
window = SC.time.now() - 30 days
total = SUM(amount) WHERE apply_date IN window
threshold = 应招标限额(物料类型)
// 物资类 ≥20万 / 工程 ≥50万 / 服务·修 ≥10万
IF total >= threshold AND apply_count > 1:
TRIGGER ALR-PUR-SPLIT-001
level = total >= threshold * 1.5 ? 'CRITICAL' : 'WARNING'
block_subsequent_split = true (新申请受阻断)
风险级别 + 处置动作
⚠ WARNING
累计 ≥ 应招标阈值 → 提醒业务部主管 + 进合规月报
🚨 CRITICAL
累计 ≥ 应招标阈值 × 1.5 → 强制合并审批（或转招标）+ 经发部审计抽查 + 阻断后续拆单
