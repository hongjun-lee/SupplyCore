🛤 链路：P-05 采购订单 ←→ S-02 入库单 ←→ F-08 发票 → 容差自动判定（数量 / 单价 / 金额 / 税率）→ 全匹配 / 差异挂账 / 拒付。
⚖ 容差：金额 ±0.5%（含税）/ 数量 ±0.5%（按计量单位）/ 单价 ±1% / 税率必须一致；超容差 → 进人工复核 → 例外审批留痕。
🛡 暂估闭环：S-02 入库无发票 → BIZ-002 暂估 → 发票到 → BIZ-003 红字冲销 + BIZ-001 正式入账（详设 06 §6.5）。
