# 物资管理系统与财务系统接口规范

**版本：** V1.2  
**日期：** 2026-03-20  
**适用范围：** 集团型煤炭企业物资供应管理系统与用友NC财务系统业务对接

---

## 一、概述

### 1.1 两系统定位与分工

| 维度 | 物资管理系统 | 用友NC财务系统 |
|------|------------|--------------|
| 核心职责 | 实物流转管理 | 资金价值核算 |
| 管理对象 | 物料、库存、出入库单据 | 凭证、科目、成本 |
| 数据粒度 | 细（按规格、批次、仓库） | 粗（按存货、科目、组织） |
| 驱动逻辑 | 业务驱动（有单据才有操作） | 凭证驱动（有凭证才记账） |
| 关系 | 业务发生方，数据源头 | 价值记录方，财务结果 |

> **核心原则：物资系统是业务的主人，财务系统是价值的镜子。**
> 物资每一笔实物流转，都应在财务留下对应的价值记录。

### 1.2 接口总体架构

```
物资管理系统
    │
    ├─ 主数据同步 ──────────→ 存货档案（N:1映射）
    │
    ├─ 合同/订单 ───────────→ 预付款登记
    ├─ 采购入库单 ──────────→ 暂估/正式入库凭证（三单匹配）
    ├─ 采购退货单 ──────────→ 红字入库冲销凭证
    ├─ 领料出库单 ──────────→ 费用/成本凭证
    ├─ 退料入库单 ──────────→ 红字费用冲销凭证
    ├─ 调拨单 ──────────────→ 内部往来/库存调整凭证
    ├─ 盘点单（盘盈）────────→ 盘盈凭证
    ├─ 盘点单（盘亏）────────→ 盘亏凭证
    ├─ 废旧处置单 ──────────→ 报废/变价收入凭证
    ├─ 质检不合格单 ─────────→ 退货或让步入账凭证
    └─ 月末结账 ────────────→ 存货计价、摊销、结转凭证
```

## 二、主数据对接

### 2.1 物料 ↔ 存货编码映射

采用N:1映射策略，详见《物资编码规范文档》第七章。

| 物资系统 | NC财务系统 | 说明 |
|---------|-----------|------|
| 物料编码（mat_code） | 存货编码（invcode） | N:1，多规格对应一存货 |
| 物料名称 | 存货名称 | 参考同步，NC为准 |
| 计量单位 | 计量单位 | 必须统一，见2.2 |
| 大类/分类 | 存货分类 | 按大类建立对应关系 |

### 2.2 计量单位统一

物资系统与NC须维护统一单位字典，禁止出现同一物料两系统单位不一致情况。

| 单位类型 | 常见单位 | 注意事项 |
|---------|---------|---------|
| 重量 | kg、t | 统一用kg或t，不混用 |
| 长度 | m、km | 电缆、风筒等 |
| 数量 | 根、个、套、台、顶 | 按物料特性定义 |
| 体积 | 桶、L | 油脂类 |
| 面积 | m² | 网片、防水材料 |

> 若换算不可避免（如采购单位与领用单位不同），在物料表维护**换算比例**字段，接口推送时自动转换。

## 三、采购业务 → 财务凭证

### 3.1 预付款/订金处理

大宗物资采购签订合同后，可能需预付定金或首付款。

**预付款凭证（合同签订/付款时）：**

| 借方 | 贷方 |
|------|------|
| 1123 预付账款 | 1002 银行存款 |

**核销（发票到达、货物入库后）：**

| 借方 | 贷方 |
|------|------|
| 2202 应付账款 | 1123 预付账款（核销预付） |
| 1403 原材料 | 2202 应付账款 |

> 物资系统须记录合同号，推送NC时携带合同信息，财务据此核销预付款。

### 3.2 三单匹配（采购订单 + 入库单 + 发票）

三单匹配是采购付款的核心控制点，三单须一致才能付款。

| 匹配状态 | 处理方式 |
|---------|---------|
| 三单一致 | 正常付款，制正式入库凭证 |
| 入库 > 订单数量 | 超出部分暂存，通知采购确认是否补单 |
| 发票金额 > 订单金额 | 价差需审批，财务记录价差科目 |
| 发票先到、货未到 | 挂暂收票据，待货物入库后匹配 |
| 货到票未到（月末）| 暂估入账，见3.3 |

### 3.3 运费/装卸费归集

采购运费是存货成本的组成部分，处理方式：

**方式A（直接计入）：** 运费金额小，直接计入对应物料成本

| 借方 | 贷方 |
|------|------|
| 1403 原材料（含运费分摊） | 2202 应付账款（货款+运费合并） |

**方式B（单独核算）：** 运费金额大或多批物料共享，单独记录后分摊

| 借方 | 贷方 |
|------|------|
| 1403 原材料-待分摊运费 | 2202 应付账款 |

> 分摊基准：按重量或金额比例分摊至各物料，调整其移动平均成本。
> **待确认：** 企业选择方式A还是方式B，需财务确定统一政策。

### 3.4 暂估入库凭证

**触发时机：** 月末，物资已入库但发票未到

| 借方 | 贷方 |
|------|------|
| 1403 原材料（暂估价） | 2181 应付暂估款 |

- 暂估价取**采购订单单价**，无订单取**上期移动平均价**
- 次月发票到达后，先红字冲销暂估，再正式入账

### 3.5 正式入库凭证

**触发时机：** 发票审核通过，与入库单匹配

| 借方 | 贷方 |
|------|------|
| 1403 原材料（发票含税价） | 2202 应付账款 |
| 2221 应交税费-进项税 | |

> 发票价与暂估价差异处理：
> - 差异 ≤5%：计入当期损益（管理费用-存货价差）
> - 差异 >5%：调整存货成本，财务审批后处理

### 3.6 供应商月末对账

每月末与供应商核对应付账款余额：
- 物资系统导出：本月采购入库汇总（含暂估）
- NC导出：应付账款明细（按供应商）
- 差异原因：发票时间差、退货未处理、价差未确认
- 对账完成后双方签字，作为付款依据

### 3.7 采购退货凭证

**触发时机：** 物资系统生成采购退货单（质量问题/数量溢出等）

| 借方 | 贷方 |
|------|------|
| 2202 应付账款（红字） | 1403 原材料（红字） |

## 四、领料出库 → 财务凭证

### 4.1 业务流程

```
使用单位提领料申请
    ↓
仓库审核，出库
    ↓
物资系统生成领料出库单（含成本中心/工程项目信息）
    ↓
推送NC → 生成费用/成本凭证
```

### 4.2 领料出库凭证

**触发时机：** 领料单审核通过，实物出库

| 借方 | 贷方 |
|------|------|
| 6602 生产成本-材料费（按成本中心） | 1403 原材料 |

- 出库单价采用**加权移动平均法**（月加权或永续加权，与NC一致）
- 领料单须携带**成本中心编码**，财务凭证据此归集费用
- 同一领料单涉及多成本中心时，凭证按成本中心分行

### 4.3 成本中心对照表

物资系统的"使用单位/用途"需与NC成本中心建立对应关系：

| 物资系统使用单位 | NC成本中心 | 说明 |
|----------------|-----------|------|
| 综采队 | CC-XX-001 | 采煤直接成本 |
| 掘进队 | CC-XX-002 | 掘进直接成本 |
| 机电科 | CC-XX-010 | 间接成本 |
| 通风区 | CC-XX-011 | 间接成本 |
| 行政部门 | CC-00-001 | 管理费用 |
| 安全部门（安全专项） | CC-SA-001 | 安全费专项 |

> 完整成本中心对照表需财务与各矿厂联合制定，列入待确认事项。

### 4.4 煤矿特有成本归集

**掘进成本 vs 采煤成本区分：**

| 领料用途 | 成本归属 | 说明 |
|---------|---------|------|
| 采煤工作面材料 | 采煤直接成本 | 锚杆、液压件等 |
| 巷道掘进材料 | 掘进直接成本 | 支护材料为主 |
| 巷道维护材料 | 维护费用 or 资本化 | 见下方说明 |
| 设备维修配件 | 设备维修费 | 归入对应设备 |

**巷道维护材料资本化判断：**
- 日常维护、小修：费用化（计入当期成本）
- 大修、改造、延伸（增加了巷道服务年限或功能）：资本化
- **待确认：** 资本化标准金额由财务与工程部门共同确定

### 4.5 退料入库凭证

**触发时机：** 使用单位退回未用完物料

| 借方 | 贷方 |
|------|------|
| 1403 原材料 | 6602 生产成本-材料费（红字冲销） |

- 退料价格取**原出库单价**，不重新计算
- 退料单须关联原领料单，财务冲销对应费用

## 五、库存调拨 → 财务凭证

### 5.1 同组织内跨仓调拨

同一核算组织内不同仓库之间的调拨，**不产生财务凭证**，仅物资系统内部库位变更。

### 5.2 跨组织调拨（矿与矿之间）

| 场景 | 凭证处理 |
|------|---------|
| 无偿调拨 | 调出方：贷1403；调入方：借1403；内部往来科目对冲 |
| 有偿调拨（内部结算） | 调出方走内部销售，调入方走内部采购，月末抵消 |

> 推荐使用无偿调拨，减少内部结算复杂度。跨组织调拨单须双方仓库签收确认。

## 六、质量管理 → 财务凭证

### 6.1 质检流程与财务节点

```
采购货物到货
    ↓
物资系统：入待检区（不入正式库存，不制凭证）
    ↓
质检结果
    ├─ 合格 → 正式入库 → 制入库凭证（见第三章）
    ├─ 部分合格 → 合格部分入库 + 不合格部分退货
    ├─ 让步接收 → 降价入库，差价冲抵应付账款
    └─ 全部不合格 → 退货，制退货凭证
```

### 6.2 让步接收降价入账

质量略有瑕疵但仍可使用，供应商同意降价：

| 借方 | 贷方 |
|------|------|
| 1403 原材料（降价后金额） | 2202 应付账款（降价后金额） |
| 2221 应交税费-进项税（按降价后） | |

> 降价协议须书面确认，附质检报告，财务留存备查。

### 6.3 质检不合格退货凭证

同采购退货处理，见3.7，附质检不合格报告。

### 6.4 合理损耗定额管理

部分物料（如炸药、油脂）在运输、储存过程中存在合理损耗：

- 物资系统维护各物料**损耗定额**（如炸药0.3%）
- 入库时系统自动计算理论损耗，在定额内的差异自动消化
- 超出定额的损耗单独列示，走盘亏流程处理（见第七章）

## 七、盘点业务 → 财务凭证

### 7.1 盘点流程

```
物资系统发起盘点任务
    ↓
仓库人员实盘，录入实盘数量
    ↓
系统计算盘盈/盘亏数量及金额（按移动平均成本）
    ↓
盘点审批（盘亏需附说明）
    ↓
推送NC → 生成盘盈/盘亏凭证
    ↓
财务审核，查明原因，月末结账前完成处理
```

### 7.2 盘盈凭证

| 借方 | 贷方 |
|------|------|
| 1403 原材料 | 6301 营业外收入-盘盈 |

> 火工品盘盈须上报，由安全部门配合核查原因，不得自行消化。

### 7.3 盘亏凭证

**正常损耗（自然损耗，在定额内）：**

| 借方 | 贷方 |
|------|------|
| 6601 管理费用-存货损失 | 1403 原材料 |

**非正常损耗（丢失/事故/超定额）：**

| 借方 | 贷方 |
|------|------|
| 6301 营业外支出-非常损失 | 1403 原材料 |

> 责任人赔偿部分：借1221其他应收款，贷6301营业外支出（冲回）。

### 7.4 呆滞库存减值

对长期未动库存（建议标准：超过12个月未出库）进行减值处理：

**识别：** 物资系统定期输出呆滞库存报表（按最后出库日期）

**财务处理：**

| 借方 | 贷方 |
|------|------|
| 6181 资产减值损失-存货跌价 | 1471 存货跌价准备 |

> 减值后物资系统库存数量不变，但账面价值降低。后续若实际出售价格高于账面，冲回减值准备。

## 八、废旧物资处置 → 财务凭证

### 8.1 废旧物资来源

- 设备维修拆下的旧配件
- 到期不可继续使用的物料（如过期炸药、老化电缆）
- 报废设备拆解后的零部件
- 生产过程中产生的废料（废钢、废旧包装）

### 8.2 物资系统处理

1. 发起废旧物资鉴定申请
2. 技术/安全部门鉴定（可修复/报废/降级使用）
3. 审批通过后转入废旧品仓
4. 处置方式：变卖/以旧换新/内部再利用/销毁

### 8.3 变卖废旧物资凭证

**出库（废旧物资出库）：**

| 借方 | 贷方 |
|------|------|
| 6301 营业外支出（账面净值） | 1403 原材料 |

**收入（变卖收款）：**

| 借方 | 贷方 |
|------|------|
| 1002 银行存款 | 6301 营业外收入（变卖价款） |

> 净额为正：营业外收入；净额为负：营业外支出。

### 8.4 以旧换新处理

供应商以旧换新（如旧截齿换新截齿）：
- 物资系统记录：旧件出库 + 新件入库（差价采购）
- 财务凭证：按差价金额处理采购入库，旧件出库按残值计入成本冲减

### 8.5 危险品销毁

过期炸药、雷管等危险品不可变卖，须按规程销毁：
- 安全部门主导，销毁过程留存记录
- 财务按账面价值全额转销毁损失

| 借方 | 贷方 |
|------|------|
| 6301 营业外支出-危险品销毁 | 1403 原材料-火工品 |

## 九、低值易耗品摊销

### 9.1 低值易耗品认定标准

单价低于固定资产标准（通常 <2000元）但使用寿命超过一年的物品，如小型工具、安全帽、矿灯等。

### 9.2 摊销方式

| 方式 | 适用场景 | 处理逻辑 |
|------|---------|---------|
| 一次性摊销 | 价值小、损耗快（如手套、口罩） | 领用时全额计费用 |
| 五五摊销 | 价值较大、有一定使用寿命 | 领用时摊50%，报废时摊50% |
| 分期摊销 | 价值较大、使用寿命明确 | 按月均摊 |

> **待确认：** 企业统一摊销政策，避免各矿自行决定造成核算不一致。

### 9.3 领用摊销凭证（一次性）

| 借方 | 贷方 |
|------|------|
| 6602 生产成本-低耗品摊销 | 1405 低值易耗品 |

### 9.4 五五摊销

**领用时（摊50%）：**

| 借方 | 贷方 |
|------|------|
| 1412 低值易耗品摊销（50%） | 1405 低值易耗品在用 |
| 6602 生产成本-低耗品摊销（50%） | 1412 低值易耗品摊销 |

## 十一、寄售/代管物资

### 11.1 定义

供应商货物存放于企业仓库，但所有权未转移，企业未付款，货物不计入企业资产。常见于大型备件寄售、试用期物资等。

### 11.2 物资系统处理

- 寄售物资单独建**寄售仓**，与正式库存隔离
- 寄售物资不参与库存金额计算，不推送财务凭证
- 领用寄售物资时才触发采购流程（所有权转移）

### 11.3 领用寄售物资凭证

寄售物资实际领用（所有权转移）时，同时制两张凭证：

**采购入库（所有权转移）：**

| 借方 | 贷方 |
|------|------|
| 1403 原材料 | 2202 应付账款 |

**领料出库（同时）：**

| 借方 | 贷方 |
|------|------|
| 6602 生产成本-材料费 | 1403 原材料 |

> 或直接合并为一张凭证，借生产成本，贷应付账款。  
> **待确认：** 合并还是分开，由财务结合NC操作习惯确定。

## 十三、存货计价与月末结账

### 13.1 计价方法

| 方法 | 说明 | 适用场景 |
|------|------|---------|
| 加权移动平均法 | 每次入库后重新计算平均成本 | 推荐，物资与NC同步计算 |
| 先进先出法（FIFO） | 批次管理物料（HG/HX） | 须启用批次管理 |

> **两系统计价方法必须一致，否则月末对账必然出现差异。**

### 13.2 月末结账流程

```
25日：物资系统停止当月补录，数据封存
    ↓
26~28日：物资系统导出月度汇总，与NC核对
    ↓
差异处理（见第十四章）
    ↓
低值易耗品摊销确认
    ↓
暂估冲销与滚动暂估
    ↓
月末最后一天：NC完成存货科目结账
    ↓
次月1日：物资系统开放次月操作
```

### 13.3 暂估冲销（月末必做）

1. 红字冲销上月全部暂估凭证
2. 对尚未收票的入库单重新暂估（滚动暂估）
3. 已收票的在发票匹配时已正式入账，不再暂估

### 13.4 存货跌价准备（年末）

年末财务部门对库存物料进行减值测试：
- 物资系统提供各物料库存数量及上次采购单价
- 财务与市场价格对比，差异>20%的计提跌价准备
- 跌价准备在NC单独处理，不影响物资系统库存数量
- 呆滞物资（超12个月未动）重点关注

## 十四、对账与差异处理

### 14.1 日常对账

| 频率 | 对账内容 | 执行方 |
|------|---------|-------|
| 每日 | 当日出入库笔数及金额 vs NC当日凭证 | 系统自动比对 |
| 每周 | 本周物资库存余额 vs NC存货科目余额 | 物资+财务 |
| 每月 | 全量库存数量/金额对账，出具对账报告 | 物资+财务 |
| 每年 | 年末存货盘点配合财务审计 | 物资+财务+审计 |

### 14.2 年末审计配合

年末外部审计时，物资系统须配合提供：
- 年末库存清单（全部物料，按仓库）
- 年度出入库明细（可导出）
- 大额收发明细（审计抽查依据）
- 盘点记录及差异处理结果
- 废旧物资处置清单

审计人员可能到仓库实地抽盘，物资系统须支持按审计指定物料导出库存记录。

### 14.3 常见差异原因及处理

| 差异原因 | 处理方式 |
|---------|---------|
| 物资已入库，NC未推送 | 检查接口推送状态，补推 |
| 单价不一致（计价方法差异） | 核实两系统计价参数，统一后重算 |
| 单位换算错误 | 核查换算比例，修正后补差额凭证 |
| 暂估未冲销 | 月末执行暂估冲销流程 |
| 手工凭证未同步 | 财务手工凭证须通知物资系统备注 |
| 跨月调整 | 需跨月补单，双方签字确认 |
| 寄售/甲供混入正式库存 | 核查仓别，调整至正确仓库 |

### 14.4 跨期单据处理

业务已发生但当月未录入（跨期）的处理原则：
- 原则上不允许跨期补录，须在当月完成
- 确需跨期的，须填写跨期申请，财务审批
- 财务NC侧需对应开放已封账期间（谨慎操作）
- 跨期单据在系统中标注，便于审计追溯

## 十五、特殊业务处理

### 15.1 火工品（HG）

- 出入库实时推送NC，不等批量
- 独立存货科目，独立凭证序号
- 月末单独出具火工品台账与NC科目余额对照表
- 任何盘亏必须附安全部门事故调查报告才能制证
- 过期危险品销毁须履行审批程序，财务全额转损失

### 15.2 大额备件资本化

单件金额超过集团规定标准（待确认，参考 ≥5万元）的备件，财务确认后按固定资产处理：

```
采购入库 → 借1405低值易耗品（临时）
    ↓
财务确认资本化
    ↓
调整凭证：借1601固定资产，贷1405低值易耗品
    ↓
物资系统标记该物料"资本化"，后续领用不制费用凭证
```

### 15.3 委外加工

物料出库送委外加工，不按领用处理：
- 物资系统：出库至"委外在制"虚拟仓
- NC：不制领用凭证，制"委外加工"发出存货科目
- 支付加工费：借1403（加工费计入存货成本），贷应付账款
- 加工完成入库后，恢复正常存货核算

### 15.4 样品/试验品领用

试验室或技术部门领用，不计入生产成本：
- 领料单用途类型标注"试验/研发"
- 财务凭证归入：研发费用 或 管理费用-技术开发

### F.4 异常场景补充

**场景A：发票一直不来怎么办？**

每个月末重复执行"冲销+重新暂估"（滚动暂估），直到发票到达：

```
4月1日: 冲销3月暂估
4月30日: 发票仍未到 → 重新暂估（暂估批次=2026-04）
5月1日: 冲销4月暂估
5月15日: 发票终于到了 → 正式入账
```

**场景B：发票价差 > 5% 怎么办？**

```
订单价 28元，发票价 32元，差异率 14.3% > 5%
  → 不自动处理，触发价差审批流程
  → 审批通过后按实际发票价调整存货成本
  → 差额 (32-28)×500 = 2000元 调整存货
  → 推送NC调整凭证
```

**场景C：到货数量与订单不一致怎么办？**

```
订单500根，实到520根
  → 超出20根暂存待处理
  → 500根正常入库暂估
  → 20根通知采购：补单 or 退回
  → 补单后20根追加入库
  → 退回则不影响
```
